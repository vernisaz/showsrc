# Source show

## Purpose
A purpose of the project to show an arbitrary source file in RDS

## Install
Copy the built CGI script to CGI directory of [RDS](https://github.com/vernisaz/rust_dev_studio)

## Build
You need to have [RustBee](https://github.com/vernisaz/rust_bee) installed and [SimpleWeb](https://github.com/vernisaz/simweb) cloned first.
The rest is quite simple:
- build simweb running `rb` in its repository
- build the plugin running `rb` here

If you prefer using Cargo, then you need to create .toml manifest first.
